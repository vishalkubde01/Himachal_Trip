📥 Offline Map Setup
Do BEFORE you lose internet:

Google Maps Setup:
1. Open Google Maps app
2. Search "Chandigarh" → Profile → Offline maps
3. Tap "Select your own map"
4. Zoom to cover all of Himachal
5. Download (~1-2 GB)

Areas to download:
✓ Chandigarh + surroundings
✓ Shimla + Mashobra
✓ Mandi + Manali (large area)
✓ Kullu Valley
✓ Dalhousie + Khajjiar
✓ Atal Tunnel + Sissu (Lahaul)
